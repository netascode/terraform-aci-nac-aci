# Terraform ACI L2 MTU Policy Module

Description

Location in GUI:
`Fabric` » `Fabric Policies` » `Policies` » `Global` » `Fabric L2 MTU`

## Examples

```hcl
module "aci_l2_mtu_policy" {
  source  = "netascode/l2-mtu-policy/aci"
  version = ">= 0.1.0"

  name          = "L2_8950"
  port_mtu_size = 8950
}
```
