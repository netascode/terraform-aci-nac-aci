# Terraform ACI VRF Module

Manages ACI VRF

Location in GUI:
`Tenants` » `XXX` » `Networking` » `VRFs`

## Examples

```hcl

  source  = "netascode/vrf/aci"
  version = ">= 0.2.4"

  tenant                                 = "ABC"
  name                                   = "VRF1"
  alias                                  = "VRF1-ALIAS"

  pim_enabled                            = true

  pim_fast_convergence                   = true
  pim_strict_rfc                         = true
  pim_max_multicast_entries              = 1000

    {
      ip                  = "1.1.1.1"
      multicast_route_map = "TEST_RM"
    },
    {
      ip = "1.1.1.2"

    {
```
