# Terraform ACI Remote Location Module

Manages ACI Remote Location

Location in GUI:
`Admin` » `Import/Export` » `Remote Locations`

## Examples

```hcl
module "aci_remote_location" {
  source  = "netascode/remote-location/aci"
  version = ">= 0.1.0"

  name          = "RL1"
  description   = "My Description"
  hostname_ip   = "1.1.1.1"
  auth_type     = "password"
  protocol      = "ftp"
  path          = "/"
  port          = 21
  username      = "user1"
  password      = "password"
  mgmt_epg_type = "oob"
  mgmt_epg_name = "OOB1"
}
```
