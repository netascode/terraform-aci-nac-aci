# Terraform ACI vPC Group Module

Manages ACI vPC Group

Location in GUI:
`Fabric` » `Access Policies` » `Policies` » `Switch` » `Virtual Port Channel default`

## Examples

```hcl
module "aci_vpc_group" {
  source  = "netascode/vpc-group/aci"
  version = ">= 0.2.0"

  mode = "explicit"
  groups = [{
    name     = "VPC101"
    id       = 101
    policy   = "VPC1"
    switch_1 = 101
    switch_2 = 102
  }]
}
```
