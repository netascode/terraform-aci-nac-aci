# Terraform ACI Maintenance Group Module

Manages ACI Maintenance Group

Location in GUI:
`Admin` » `Firmware` » `Nodes`

## Examples

```hcl
module "aci_maintenance_group" {
  source  = "netascode/maintenance-group/aci"
  version = ">= 0.1.0"

  name     = "UG1"
  node_ids = [101]
}
```
