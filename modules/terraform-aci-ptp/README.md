# Terraform ACI PTP Module

Manages ACI PTP

Location in GUI:
`System` » `System Settings` » `PTP and Latency Measurement`

## Examples

```hcl
module "aci_ptp" {
  source  = "netascode/ptp/aci"
  version = ">= 0.1.1"

  admin_state       = true
  global_domain     = 0
  profile           = "aes67"
  announce_interval = 1
  announce_timeout  = 3
  sync_interval     = -3
  delay_interval    = -2
}
```
